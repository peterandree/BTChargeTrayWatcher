# iOS Installation Guide — ntfy Push Notifications

This guide explains how to install the ntfy app on iPhone and subscribe to BTChargeTrayWatcher battery alerts.

## Prerequisites
- An iPhone running iOS 15 or later.
- BTChargeTrayWatcher installed and running on your Windows PC.
- A topic generated by BTChargeTrayWatcher (see **Mobile notifications → Show topic** in the tray menu).

## Steps

### 1 — Install ntfy
1. Open the **App Store** on your iPhone.
2. Search for **ntfy**.
3. Install the app by Philipp C. Heckel.

### 2 — Subscribe to your topic
1. Open the ntfy app.
2. Tap **+** or **Add subscription**.
3. For **Server**, enter: `ntfy.sh`
4. For **Topic**, enter the topic shown by BTChargeTrayWatcher.
   - You can find it under **Mobile notifications → Show topic** in the tray menu.
   - Example: `btcw-8h2qv6m4k9r1x7p3`
5. Tap **Subscribe**.

### 3 — Allow notifications
1. When prompted by iOS, tap **Allow**.
2. If the prompt does not appear, go to **iPhone Settings → Notifications → ntfy** and enable **Allow Notifications**.
3. Ensure **Time Sensitive Notifications** is also enabled for urgent battery alerts.

### 4 — Test the setup
1. Return to BTChargeTrayWatcher on your PC.
2. Open **Mobile notifications → Send test notification** in the tray menu.
3. Confirm the notification arrives on your iPhone within a few seconds.

## Notes
- iOS delivers ntfy push notifications via Apple Push Notification service (APNs). The public ntfy.sh server handles this relay automatically — no additional configuration is required.
- The ntfy app does not need to be open to receive notifications.

## Troubleshooting
- **No notification received:** Verify the topic matches exactly. Topics are case-sensitive.
- **Notifications appear only when app is open:** Ensure notifications are allowed in iPhone Settings.
- **Notifications delayed:** This can occur if iOS Focus Mode or Do Not Disturb is active. Check your Focus settings.
